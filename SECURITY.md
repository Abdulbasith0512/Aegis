# Security Policy

## Supported Versions

Only the latest active release version of AegisAI receives security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of AegisAI seriously. If you find a security vulnerability, please do NOT create a public GitHub issue. Instead, report it privately.

Please email security reports to **security@aegisai.org** with:
- A clear description of the vulnerability.
- Steps or scripts to reproduce the issue.
- Potential impact or exploitation details.

Our team will acknowledge receipt of your report within 48 hours and work with you to coordinate a secure release fix.
